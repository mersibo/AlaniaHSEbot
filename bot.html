async bot.add_allowed_user(message: Message)[исходный код]¶

Добавляет нового пользователя в список разрешённых пользователей для выполнения определённых команд.

Команда доступна только администраторам. После выполнения добавляет указанного пользователя в список разрешённых, если он ещё не добавлен.

Аргументы: message (types.Message): Сообщение с командой, в котором указан ID пользователя.

Исключения: Отправляет сообщение об ошибке, если формат команды неверен или если пользователь уже добавлен.

async bot.export_excel(callback: CallbackQuery)[исходный код]¶

Экспортирует данные пользователей в Excel и отправляет файл пользователю, если он зарегистрирован.

Эта функция выполняет следующие действия: 1. Извлекает данные пользователей из базы данных. 2. Формирует DataFrame с полученными данными и сохраняет его в Excel-файл. 3. Применяет стили к заголовкам и данным в таблице (шрифт, выравнивание, ширина столбцов). 4. Проверяет, зарегистрирован ли пользователь, инициировавший запрос, в базе данных. 5. Если пользователь зарегистрирован, отправляет ему Excel-файл. 6. Если нет, выводит сообщение, что необходимо зарегистрироваться.

Аргументы: callback (CallbackQuery): Объект CallbackQuery, содержащий информацию о запросе от пользователя.

Исключения: None

Аргументы: message (Message): Сообщение, которое нужно проверить на наличие нецензурных слов.

async bot.handle_broadcast_message(message: Message, state: FSMContext)[исходный код]¶

Обрабатывает сообщение для рассылки.

После выбора группы пользователь отправляет сообщение, которое бот рассылает указанной группе. Поддерживаются текст, фото и видео.

Аргументы: message (types.Message): Сообщение от пользователя с контентом для рассылки. state (FSMContext): Контекст состояния FSM для извлечения данных о выбранной группе.

async bot.start_broadcast(message: Message)[исходный код]¶

Запускает процесс выбора группы для рассылки сообщения.

Команда доступна только разрешённым пользователям и администраторам. Функция выводит список доступных групп для рассылки с использованием инлайн-кнопок.

Аргументы: message (types.Message): Сообщение от пользователя с командой.

Исключения: Отправляет сообщение, если список групп пуст или если у пользователя нет прав для рассылки.

async bot.tag_admins(message: Message)[исходный код]¶

Упоминает всех администраторов группы.

Эта команда генерирует список упоминаний всех администраторов группы и отправляет их в сообщении.

async bot.tag_all(message: Message)[исходный код]¶

Упоминает всех участников группы, кроме ботов.

Эта команда доступна только администраторам. Она генерирует список упоминаний всех участников группы и отправляет их в несколько сообщений, если количество упоминаний превышает лимит.

Аргументы: message (types.Message): Сообщение от администратора группы с командой.

Исключения: Отправляет сообщение об ошибке, если пользователь не является администратором.
